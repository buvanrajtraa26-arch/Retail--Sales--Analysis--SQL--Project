🛒Retail Sales Analysis using MySQL

📌Project Overview

This project focuses on analyzing retail sales data using MySQL. The objective is to derive business insights related to sales performance, customer behavior, product demand, and city-wise revenue generation.

🛠️ Tools Used
- MySQL Workbench
- SQL

🗄️ Database

Retail_Sales_Analysis_Project

📂Dataset Details
The dataset contains:
- Order ID
- Order Date
- Customer Name
- City
- Product
- Category
- Quantity
- Price



 📊 Business Problems Solved

✅  Total Number of Orders
Calculated the total number of orders processed in the dataset.

✅  Total Revenue Analysis
Computed overall sales revenue using quantity and product price.

✅  Category-wise Sales Performance
Analyzed total revenue generated by each product category.

✅ City-wise Sales Analysis
Evaluated sales performance across different cities.

✅  Top-Selling Product
Identified the product with the highest quantity sold.

✅  Highest Value Order
Determined the order generating the maximum sales value.

✅ Average Order Value
Calculated average spending per order.

✅  Customer Spending Analysis
Identified high-value customers based on total purchase amount.

✅  Monthly Sales Trend Analysis
Analyzed revenue trends over time using monthly sales data.

✅  Customer Distribution by City
Counted the number of orders from each city.

✅  High Revenue Categories
Identified categories generating sales greater than a specified threshold.
